# Revised Battles
Revised Battle is a mod that enhances many battles, either by boosting enemies or adding new ones. This transforms some pathetic fights into bloody or even brutal massacres. All components are optional, except for the first (detectable spell) and the last (patch for all files installed by the mod).

The mod also offers the installation of 4 warrior kits, 3 of which are very cool (the last one is kind of a joke):
 - Avenger of the arcane
 - Master-at-arms
 - Demon Lord
 - Bastard
 
 
Notes :
 - This mod adds powerful weapons to enemies, which they will drop. This is one of the main criticisms leveled at this mod: the abundance of magical weapons.
 - Since this mod improves enemies by boosting their level, re-equipping them or adding followers, it is recommended to install it before SCS, BP or any other mods that improve AI / enemies.
